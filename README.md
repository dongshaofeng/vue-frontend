# vue-frontend  
Vue frontend Admin UI , support Iview and ElementUI .   


# How to Run 
cd vue-fronted  
npm install  
npm run dev  

# How to build 
npm run build:prod

# --preview to start a server in local to preview
npm run build:prod --preview

# lint code
npm run lint

# auto fix
npm run lint -- --fix

# visit
http://localhost:9527
